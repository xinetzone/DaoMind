## 🎥 录制建议

如果你想基于这些脚本录制视频：

### 技术要求
- **录屏软件**: OBS Studio / ScreenFlow / Camtasia
- **代码编辑器**: VS Code（推荐主题：One Dark Pro）
- **字体**: Fira Code / JetBrains Mono（启用连字）
- **分辨率**: 1920x1080 (16:9)
- **帧率**: 30fps 或 60fps

### 录制清单
- [ ] 准备干净的 VS Code 工作区
- [ ] 关闭不必要的通知
- [ ] 准备好所有代码示例
- [ ] 测试麦克风和录屏
- [ ] 准备饮用水
- [ ] 检查时长控制

### 后期制作
- **剪辑**: DaVinci Resolve / Final Cut Pro / Premiere Pro
- **字幕**: 使用自动字幕 + 手动校对
- **封面**: 统一风格，包含集数和标题
- **音乐**: 轻柔的背景音乐（如需要）
