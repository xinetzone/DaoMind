## 五、关键技术经验

### 1. EventEmitter 泄漏防护（ObserverAgent）
```typescript
// 构造函数存储绑定引用，initialize/terminate 精确管理注册
constructor(id: string) {
  super(id);
  this.boundHandler = (event) => { this.observations.push(...); };
}
override async initialize() { daoNothingVoid.on('observed', this.boundHandler); }
override async terminate() { daoNothingVoid.removeListener('observed', this.boundHandler); }
```

### 2. 测试顺序 — 避免跨测试污染
```typescript
beforeEach(() => {
  daoNothingVoid.void();         // 必须第一步清空监听器
  universe = new DaoUniverse();  // 然后再创建新实例
});
afterEach(() => { universe.bridge.dispose(); });
```

### 3. TypeScript 隐式 number 返回
```typescript
// 错误：Array.push() 返回 number，与 MessageHandler void 不兼容
agent.onMessage((m) => arr.push(m));
// 正确：块体显式 void
agent.onMessage((m) => { arr.push(m); });
```

### 4. tsconfig project references
```json
{ "composite": true, "references": [{"path": "../daoNothing"}, ...] }
// build 必须用 tsc --build，不能用 tsc
```
