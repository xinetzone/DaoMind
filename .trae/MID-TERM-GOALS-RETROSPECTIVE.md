```typescript
export class UserService {
```

#### 技术亮点

1. **架构设计**:
   ```
   UserService (业务逻辑层)
   ├── PermissionService (权限服务)
   ├── UserRepository (数据仓库模式)
   └── ValidationHelpers (验证工具)
   ```

2. **安全最佳实践**:
   - 密码哈希（不存储明文）
   - 输入验证（防止注入攻击）
   - 权限检查（最小权限原则）

   ```typescript
   private emailIndex: Map<string, string>;      // email -> id

   ```

4. **不可变更新**:
   ```typescript
   const updated: User = {
     ...user,
     ...input,
   };
   ```
