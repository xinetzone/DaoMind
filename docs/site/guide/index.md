# 文档介绍

欢迎使用 **DaoMind & Modulux** —— 一个融合道家哲学与现代 TypeScript 的模块化框架。

## 什么是 DaoMind?

DaoMind 以帛书《道德经》中的哲学概念为核心：

> **无名，万物之始；有名，万物之母。**

这句话映射到 TypeScript 的世界：

| 哲学概念 | TypeScript 对应 | 特点 |
|---------|----------------|------|
| **无名 (Wúmíng)** | 类型空间 (Type Space) | 编译时存在，运行时消失，零开销 |
| **有名 (Yǒumíng)** | 值空间 (Value Space) | 运行时实例，真实存在的模块 |

## 核心特性

- **零运行时开销** — 类型定义编译后完全消失
- **类型安全** — 编译时捕获所有错误
- **模块化架构** — 清晰的依赖管理
- **哲学驱动** — 有意义的命名和设计
